AI Resume Analyzer

AI Resume Analyzer is a full-stack web application that helps users upload their resumes, analyze resume content, and get job recommendations based on extracted skills and profile matching.
The project is built with a React frontend and a FastAPI backend, with MongoDB used for data storage. It provides user authentication, resume upload, resume parsing, and job matching features in a simple and user-friendly interface.
## Features
- User registration and login
- Secure password hashing
- JWT-based authentication
- Resume upload functionality
- Resume text extraction
- AI-based job matching
- Dashboard for analyzing resumes
- Modern frontend interface
- MongoDB integration for storing users and resumes
